Actividades

A continuación, se describen las actividades realizadas mostrando algunos procesos, productos (capas) y breve descripción de los mismos.

  • Descarga y adecuación cartográfica de las capas de variables climáticas y fisiográficas.

  • Carga de capas de puntos representativos de los productores agrícolas pertinentes a cada uno del cultivo analizado, y su sobre posición sobre la capa del índice ETO/Precipitación.

  • Extracción de los valores de cada una de las variables climofisiograficas para cada punto de los puntos de productores agrícolas.

  • Calculo de los valores de percentiles 0.05 y 0.95 de cada variable climofisiografica para la ubicación de productores pertinentes a cada cultivo.

  • Selección de las celdas de cada variable climofisiografica que cumpla el criterio de encontrarse entre los percentiles indicados.

  • Análisis celda a celda para compilar cuales celdas cumplan el criterio anterior y así forme parte de la Región de exposición de cada cultivo analizado en este proyecto.

  • Revisión bibliográfica establecer cuales cumplan

Variables climofisiograficas

Las variables climofisiograficas consideradas para el analisis segun los criterios del Plan de trabajo de UGP-PNA son:

  • Indice ET)/Preciptacion media anual.

  • Periodo de crecimiento (dias)

  • Periodo seco (dias)

  • Periodo humedo (dias)

  • Pendiente del terreno (%)

  • Altura del terreno (msnmm)
